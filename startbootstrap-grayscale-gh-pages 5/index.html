Project: Copier Tool Automation

Built copier tool in Python to automate the process of copying a template course into a permanent course in the Canvas for the Missionary Training Center. Called the Canvas API to make automations in the language courses, reducing our copying process time by half.

Project: BCC Bot (Greg)

Created automation in Python to send out survey links, reminders, and office events. Integrates with messaging platforms such as Microsoft Teams to send out messages at a specific time every week. Creates a database to store employee details, survey responses, and event RSVPs. User-friendly interface for employees to interact with the bot and update their information or preferences.

Project: Collecting and Analyzing Survey Data on Recruiter Satisfaction

Created automation that imports data from two Qualtrics reports on recruiter satisfaction surveys, manipulates the data to extract relevant information, and saves the resulting dataframe to a Google Sheet. The code uses several libraries to achieve this goal, including pandas for data manipulation, gspread to interface with Google Sheets, and dropbox to download the survey reports. Specifically, the code downloads the two survey reports from Dropbox, converts them into pandas dataframes, cleans the data, and extracts relevant columns of data. It then saves the resulting dataframe to a Google Sheet.

Project: Droplets Automation

Create automation that sends emails containing Droplets (customized messages from BCC managers and colleagues) to designated recipients. The script connects to a MySQL database, reads Droplet information from a table, creates an email for each recipient, and sends the email to the corresponding email address.
